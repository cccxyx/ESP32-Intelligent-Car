# ESP32 蓝牙遥控四驱小车（4 电机 + TB6612FNG 驱动）

## 1. 作品简介
本作品是一台基于 **ESP32** 的蓝牙遥控四驱小车。手机通过蓝牙串口向 ESP32 发送单字符指令，ESP32 输出 PWM 与方向控制信号驱动 **TB6612FNG（你称 TB1266）双路直流电机驱动板**。本车使用 **四个直流减速电机**（四驱），通过 TB6612 的 A/B 两路分别控制左右两侧，实现前进、后退、原地左/右转、停止与 0~9 档调速。

作品特点：
- 蓝牙遥控，操作简单
- PWM 调速（0~9 档）
- 左右独立驱动，支持原地转向
- 双电源供电：ESP32 与电机电源分离，稳定性更好

## 2. 基本功能
### 2.1 指令说明（手机 APP → 蓝牙）
- `F`：前进
- `B`：后退
- `L`：左转（原地：左侧后退、右侧前进）
- `R`：右转（原地：左侧前进、右侧后退）
- `S`：停止（滑行停止）
- `0`~`9`：速度档位（越大越快）

### 2.2 四电机驱动方式（左右并联）
由于 TB6612 只有两路输出（A/B），本车采用“四驱左右并联”：
- **左前电机 + 左后电机 并联 → 接到 A01/A02**
- **右前电机 + 右后电机 并联 → 接到 B01/B02**

> 并联要点：同侧两个电机的“正负方向”要保持一致，否则会出现同侧两电机互相对抗、发热、没劲或不转。

## 3. 使用到的电子器件（BOM）
- ESP32 开发板（Arduino 框架）
- TB6612FNG 双路直流电机驱动模块（常见版本引脚：PWMA/AIN1/AIN2/A01/A02，PWMB/BIN1/BIN2/B01/B02，STBY，VM，VCC，GND）
- 直流减速电机 ×4
- 四驱底盘 + 车轮 ×4
- 杜邦线/端子线若干

供电：
- 充电宝：给 ESP32 通过 USB 供电
- 4 节电池盒：每格 1.5V（合计约 **6V**）给 TB6612 的电机电源 VM 供电

建议（可选）：
- 在 TB6612 的 VM-GND 之间加电解电容 470µF~1000µF（抑制电机启动压降与干扰）

## 4. 供电与共地（非常重要）
本项目是双电源：
- **充电宝 → ESP32 USB 供电**
- **电池盒（约 6V）→ TB6612 的 VM 供电**

必须做到：
- **ESP32 GND ↔ TB6612 GND ↔ 电池盒负极(GND) 三者共地**
否则控制信号没有共同参考地，会导致电机不工作或行为异常。

## 5. 电路板连线（接线图：表格版）
### 5.1 ESP32 ↔ TB6612 控制线（按当前代码）
| 功能 | TB6612 引脚 | ESP32 引脚 |
|---|---|---|
| 使能 | STBY | GPIO23 |
| A 通道 PWM | PWMA | GPIO13 |
| A 通道方向 1 | AIN1 | GPIO14 |
| A 通道方向 2 | AIN2 | GPIO27 |
| B 通道 PWM | PWMB | GPIO12 |
| B 通道方向 1 | BIN1 | GPIO26 |
| B 通道方向 2 | BIN2 | GPIO25 |
| 逻辑电源 | VCC | 3V3 |
| 地 | GND | GND（与电池盒负极共地） |

### 5.2 电机输出（四电机并联）
| 侧边 | TB6612 输出端 | 连接方式 |
|---|---|---|
| 左侧（左前+左后） | A01 / A02 | 左前与左后并联后接入 |
| 右侧（右前+右后） | B01 / B02 | 右前与右后并联后接入 |

并联接法示例（以左侧为例）：
- 将 **左前电机的两根线** 与 **左后电机的两根线** 分别“同名并到一起”
- 并联后的两根线再接到 A01/A02

### 5.3 电源线
| 电源 | 连接到哪里 |
|---|---|
| 充电宝（USB） | ESP32 USB 口 |
| 电池盒正极（约 6V） | TB6612 VM |
| 电池盒负极 | TB6612 GND |
| ESP32 GND | TB6612 GND（共地） |
| ESP32 3V3 | TB6612 VCC |

## 6. 使用方法
1. 按第 5 章完成接线，确认共地正确。
2. 上电顺序建议：
   1) 先给 ESP32 上电（充电宝 USB）
   2) 再给 TB6612 上电（电池盒接入 VM）
3. 手机蓝牙串口 APP 连接设备：`ESP32_CAR`
4. 发送指令：
   - 先发 `9`（最大速度）
   - 再发 `F/B/L/R`
   - 发 `S` 停止

## 7. 常见问题与排查
### 7.1 一侧两电机不转 / 只有一侧转
- 检查该侧是否接在正确输出端（A01/A02 或 B01/B02）
- 检查控制线是否插错到输出端（曾出现把控制线插到 B01/B02 的情况）
- 检查 STBY 是否为高电平（GPIO23 或直连 3V3）
- 检查并联时是否“同侧两电机接反导致对抗”（会出现发热、没力、抖动）

### 7.2 小车前进跑偏、左右速度不一致
- 机械差异/摩擦差异很常见
- 可通过降低速度、调整轮胎、检查卡滞
- 或在代码中做左右差速补偿（后续可扩展）

### 7.3 方向相反
- 若某侧方向反了：对调该侧输出端两根线（A01↔A02 或 B01↔B02）
- 注意：对调只会“反转方向”，不会提升扭矩或解决供电不足

### 7.4 发指令蓝牙断开/不稳定
通常是电机启动瞬间电流导致干扰或压降：
- VM-GND 加 470µF~1000µF 电容
- 电机电源线尽量短粗
- 确保共地牢靠，不要靠松动的面包板接触点

## 8. 软件说明（摘要）
- ESP32 使用 LEDC 硬件 PWM：频率 20kHz，分辨率 8bit（0~255）
- 蓝牙使用 `BluetoothSerial`
- 指令解析：`F/B/L/R/S` 与 `0~9` 调速
